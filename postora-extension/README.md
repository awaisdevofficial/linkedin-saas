# POSTORA LinkedIn Connector

Securely links your LinkedIn account to POSTORA for automated posting and engagement.

## Local Development Setup

1. Open Chrome → `chrome://extensions`
2. Enable **Developer Mode** (top right)
3. Click **Load Unpacked** → select `postora-extension/` folder
4. Log into LinkedIn in your browser
5. Open POSTORA and click **Connect LinkedIn**

## Icons

Replace `icon16.png`, `icon48.png`, `icon128.png` with brand assets before Chrome Web Store submission.
